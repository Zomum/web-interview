# mouseenter和mouseover的区别 参考

单独一个元素没有区别，但是有了子元素会有区别

由于mouseenter不支持事件冒泡，导致在一个元素的子元素上进入或离开的时候会触发其mouseover和mouseout事件，但是却不会触发mouseenter和mouseleave事件

# 用正则表达式匹配字符串，以字母开头，后面是数字、字符串或者下划线，长度为9-20

//判断字符串是否是这样组成的，第一个必须是字母，后面可以是字母、数字、下划线，总长度为9-20
var c =  /^[a-zA-Z]\w{9,20}$/;
//  /是转义   ^  是开头   [a-zA-Z]是字母出现一次    \w是任意字符   {9,20}是长度 $ 是结束
console.log(c.test()); // true

# alert(1&&2),alert(1||0) 输出什么

1、只要“||”前面为false，无论“||”后面是true还是false，结果都返回“||”后面的值。 2、只要“||”前面为true，无论“||”后面是true还是false，结果都返回“||”前面的值。 3、只要“&&”前面是false，无论“&&”后面是true还是false，结果都将返“&&”前面的值; 4、只要“&&”前面是true，无论“&&”后面是true还是false，结果都将返“&&”后面的值; 由以上可知，逻辑运算符，“||”和“&&”都是遵行短路原则，只要确定符号前面的真假，既可确定返回值。

输出 2 ， 1
